🌊 Maritime Ontology

Ontologie Juridique et Écologique des Espèces Marines Protégées

📊 Architecture du Pipeline

Le pipeline est un processus automatisé qui transforme les données brutes en une ontologie OWL complète et validée. Voici les étapes :

1 Import LKIF Stub

Entrée : data/input/lkif_stub.ttl

Fichier Python : loader.py

Sortie : Foundation ontologique LKIF (Legal Knowledge Interchange Format) chargée en mémoire

Charge les concepts juridiques de base (Regulation, Permission, Prohibition, etc.)

2 Construction du Schéma OWL

Fichier Python : schema.py

Résultat : 58 classes OWL, 22 propriétés objet, 10 propriétés données

Définit :

  • Classes principales : Interdiction, EspeceMarine, Zone, Acteur
  • Propriétés : protegeEspece, appliesInZone, concerneActeur
  • Restrictions OWL 2.0 : domaines, portées, cardinalités

3 Chargement des Données

Entrées : 6 fichiers JSON d'extraction IA (résolutions IWC, convention ICRW, etc.)

Fichier Python : loader.py

Résultat : 55 triplets bruts, 70 entités, 103 termes glossaire

Sources :

  • Resolution 61-105 (1986)
  • Resolution 64-72 (1984)
  • Resolution 66-68 (2018)
  • Resolution 71-123 (2022)
  • ICRW Convention (1946)
  • Chalutage de fond (extraction IA)

4 Population des Individus

Fichier Python : populator.py

Actions :

  • Création de 15 espèces marines (Baleine bleue, Baleine grise, Cachalot, etc.)
  • Création de 8 zones de protection (Sanctuaires océaniques, hémisphères)
  • Création de 6 sources juridiques (Résolutions, Conventions)
  • Déclaration de 8 acteurs (Pays membres IWC, navires, etc.)
  • Création de 2 interdictions : I001 (chalutage de fond) et I002 (chasse commerciale)
  • Intégration de 127 concepts lexicaux (glossaire SKOS)

4.5 Corrections & Enrichissements Post-Traitement

Fichier Python : corrections.py

10 corrections appliquées automatiquement :

  1. Correction des fautes de frappe dans les URIs (BaeleineSeI → BaleineSei)
  2. Enrichissement des sources juridiques (sourceYear, skos:definition bilinguee)
  3. Correction des définitions tronquées
  4. Ajout de rdfs:range aux propriétés
  5. Création d'individus Permission pour les exceptions scientifiques
  6. Ajout de skos:inScheme aux espèces marines
  7. Correction des références mortes
  8. Suppression des 13 individus Concept_* redondants
  9. Correction des URIs mal formées
  10. Vérification de l'intégrité XML

5 Export des Fichiers Ontologiques

Fichier Python : pipeline.py

Formats générés :

  • OWL (RDF/XML) : maritime_ontology.owl - Format standard OWL 2.0
  • Turtle : maritime_ontology.ttl - Format textuel optimisé
  • JSON-LD : maritime_ontology.jsonld - Format web-friendly
  • N-Triples : maritime_ontology.nt - Format simplifié

6 Validation SPARQL

Fichier Python : sparql_runner.py

Questions de compétence validées : 12 requêtes SPARQL

Exemples de questions :

  • CQ1 : Liste des interdictions → 2 résultats
  • CQ6 : Espèces protégées → 15 résultats
  • CQ4 : Acteurs concernés → 8 résultats

7 Export Neo4j

Fichier Python : neo4j_export.py

Résultat : Script Cypher pour importer dans Neo4j

Graphe extrait : 164 nœuds, 44 relations

8 Génération des Rapports

Fichiers générés :

  • ontology_stats.json : Statistiques détaillées
  • ontology_validation_report.json : Rapport de validation
  • sparql_results.json : Résultats des questions de compétence

📁 Structure des Fichiers Python

  • main.py : Point d'entrée principal - exécute le pipeline complet
  • pipeline.py : Orchestrateur - coordonne les 8 étapes
  • loader.py : Chargement des données LKIF et JSON
  • schema.py : Définition du schéma OWL (classes, propriétés)
  • populator.py : Création des individus et population du graphe
  • corrections.py : Post-traitement et correction des données
  • sparql_runner.py : Exécution des questions de compétence SPARQL
  • neo4j_export.py : Génération du script d'import Neo4j

cd /path/to/maritime_ontology python main.py

🗺️ Zones d'Application (4)

L'interdiction s'applique dans les zones suivantes :

  • Hémisphère sud pour les baleines : Zone principale de migration hivernale
  • Sanctuaire océan austral : Zone protégée depuis 1994
  • Sanctuaire océan indien : Zone sanctuarisée en 1999
  • HM (Indian Ocean Sanctuary) : Zone supplémentaire de conservation

👥 Acteurs Concernés (5)

Les acteurs suivants sont soumis à l'interdiction :

  • Navire usine : Interdiction d'opérations de transformation en mer
  • Bateau chasseur : Interdiction de chasse commerciale
  • Station terrestre : Interdiction de traitement commercial
  • Membre IWC : États membres de la Commission baleinière internationale
  • Factory ship : Navire de transformation spécialisé

⚖️ Activités et Stocks Concernés

L'interdiction I002 différencie clairement l'activité du stock halieutique :

Propriété Valeur Signification
concerneActivite Activite_COMMERCIAL_WHALING L'activité interdite est la chasse commerciale à la baleine
concerneStock Stock_TOUS_LES_STOCKS S'applique à tous les stocks sans distinction génétique (moratoire 1986)
protegeEspece 13 espèces (baleine franche, cachalot, orque, etc.) Les espèces protégées que l'interdiction défend

Note : Avant 1986, les quotas étaient différenciés par stock. Depuis le moratoire de 1986, l'interdiction de chasse commerciale s'applique uniformément à tous les stocks sans distinction génétique.

🕐 Période d'Application

Période : Moratoire depuis 1986
Statut : Indéfini (en vigueur depuis 1986 sans date d'expiration fixée)
Révision : Possible lors des sessions annuelles de l'IWC

🔐 Exceptions et Dérogations

Malgré l'interdiction, deux exceptions existent :

1️⃣ Exception : Chasse Scientifique (Article 8 de la Convention ICRW)

  • Permission : Permission_ChasseScientiqueArticle8
  • Contrôle : Controle_ChasseScientiqueIRWC
  • Pays utilisant : Japon, Islande (historiquement)
  • Quotas : Définis annuellement par l'IWC

2️⃣ Exception : Chasse de Subsistance Autochtone

  • Permission : Permission_ChasseSousistanceAutochtone
  • Contrôle : Controle_QuotasSubsistanceAutochtone
  • Peuples autorisés : Inuits (Canada, Groenland), Tchukches (Russie)
  • Justification : Traditions culturelles, subsistance alimentaire

📚 Fondements Juridiques

L'interdiction I002 s'appuie sur :

  • ICRW Convention (1946) : Convention internationale pour la réglementation de la chasse à la baleine
  • IWC Schedule : Calendrier des stocks de baleines et des limitations
